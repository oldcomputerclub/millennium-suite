プライバシー保護機能を使う

あなたのブラウザには、あなたの個人情報であるプライバシーやセキュリティーを確保する機能が備わっています。以下のセクションでは、インターネットを利用している間、あなたのブラウザを使って、cookie、パスワード、フォーム、そして画像を管理する方法を解説します。

Web サイトの中には、P3P（Platform for Privacy Preferences）規格に従ったプライバシーポリシーを公開しているところがあります。この規格を基に、あなたのブラウザを変更するための情報は、プライバシーレベルの設定をご覧ください。

Mozilla のセキュリティー機能についての情報は、メッセージに署名し、暗号化するや証明書を利用するをご覧ください。

ウィルス、安全な電子メール、安全なオンラインショッピングや銀行取引、そして子供のための安全な Web サーフィンなどのプライバシーやセキュリティについての一般的な情報は、オンラインドキュメントであるプライバシーを理解するをご覧ください。

ブラウザはどういう情報を Web サイトに渡しているのですか？

あなたのブラウザが Web ページを表示する時（例えば、あなたがリンクをクリックしたり、URL を入力したりする時、あるいは、Web ページが電子メールの中に表示される時）、ブラウザはそのサイトにある種の情報を渡しています。こうした情報には、あなたの利用環境、あなたのインターネットアドレス、そしてあなたがどのページから来たのか、が含まれます（ただし、これだけには限りません）。

 

利用環境

サイトは、あなたのブラウザの種類やオペレーティングシステムといったあなたの利用環境についての情報を受け取ります。この情報によってサイトは、あなたが要求しているページをスクリーン上に最適の状態で表示できるようになります。例をあげると、サイトはあなたが Netscape 6.2.3 のフランス語版を、 Windows 2000 コンピュータで利用している、という情報を得るのです。

 

インターネットアドレス

ブラウザはサイト側にインターネットアドレス （インターネットプロトコルまたは IP アドレスとしても知られています） を教えなければなりません。この情報でサイトは、あなたが要求しているページをどこに送ればいいのかが判るのです。あなたの IP アドレスが判からないと、サイトはページを表示できません。

あなたの IP アドレスは一時的なものか、固定された（静的な）ものかのどちらかです。

電話線に繋がれた標準的なモデムでインターネットに接続している場合、あなたのインターネットサービスプロバイダ （ISP） が、あなたがログオンする度に、一時的な IP アドレスを割り当てている可能性があります。あなたは、その一時的な IP アドレスを、インターネット中に使用しているのです。つまり、例えば、あなたがサインオフ、ダイアルアップ接続を切った場合、あるいはインターネットに接続したままコンピュータを終了するまでの間のことです。それぞれの ISP は沢山の IP アドレスを持っており、ユーザーに対してランダムにアドレスを割り当てています。

DSL、ケーブルモデム、または光ケーブル接続を行っている場合、インターネットにつなぐ度に固定された IP アドレスを使っている可能性があります。

あなたの IP アドレスは、あなたの電子メールアドレスとは違います。

 

どのページから要求を送っているか

サイトは、あなたがページのリンクをクリックした時に、そのサイトのどのページを読んでいたのか、という情報を得ます。これによりサイトは、どのサイトがあなたにそのページを教えたのかが判ります。またこの情報から、あなたがサイト内を動き回る際にも、あなたがそのサイトのどの部分にいたのかが判るようになるのです。

 

cookie とは何ですか？どのように使われるのでしょう？

cookie とはいくつかの Web サイトによって利用される少量の情報です。あなたがそうしたサイトを訪れた時、サイトはあなたのブラウザに、一つまたは複数の cookie をあなたのコンピュータのハードディスクに保存するように要請することがあります。

その後あなたがそのサイトに戻ったとき、あなたがブラウザはその cookie をサイトに送り返します。

標準の cookie 設定を使用している場合、このやり取りは、あなたには分かりません。また、あなたには、いつサイトが cookie を設定したのか、また、いつあなたのブラウザが cookie をサイト側に送り返しているかも分かりません。ただし、cookie が設定される前に警告を表示するように、設定を変更することができます。この設定を行う方法は、プライバシーレベルの設定をご覧ください。

 

サイトは cookie の情報をどう利用するのですか？

cookie は、あなたが前回の訪問についての情報を、サイト側に知らせるためのものです。例えば、あなたが普段、地元の天気情報や、Web サイトで本を買った場合、そのサイトは cookie を使って、あなたがどの都市に住んでいるのか、どの作者が好きなのかを記録して、あなたが次回そのサイトを訪れた時、より便利になっているのです。

Web サイトの多くが、受け取った情報をどのように使用するかを説明した、プライバシーポリシーを公開しています。

Web サイトのプライバシーポリシーを確認して、サイトが収集した情報について何をいっているのか、収集された情報がどう利用されるのか、そして収集された情報の使用に関してあなたはどのような選択肢を持っているのか、を確認してください。こうしたことをする方法は、プライバシーポリシーを表示するをご覧ください。

公開されているプライバシーポリシーを利用して、cookie を管理する方法についての情報は、プライバシーレベルの設定をご覧ください。

 

第三者 Cookie とは何ですか？

時には、Web サイトは、別の Web サイトにホストされている内容を表示することがあります。こうした内容は、画像やテキスト、または広告にまで至ります。こうした要素をホストする Web サイトはまた、あなたがそのサイトに直接行っていないにも拘わらず、あなたのブラウザに cookie を設定することができます。

あなたが訪れているサイトとは別のサイトによって保存される cookie は、第三者 cookie または外部 cookie と呼びます。Web サイトは、時に透過 GIF を使って第三者 cookie を使用します。これは、サイトがユーザーをカウントしたり、電子メールの送信を追跡したり、訪問者がそのサイトをどう利用しているのかを詳しく知るため、あるいは、あなたのブラウジングをカスタマイズするために使用される、特殊な画像です。（透過 GIF はまた、Web 上の標識灯、あるいは Web バグとしても知られています。）

あなたが望むなら、cookie 設定を変更して、サイトが普通の cookie は保存できても、第三者 cookie は保存できないようにすることもできます。

 

電子メールメッセージの中の Web ページはどのように管理できるのですか？

電子メールメッセージの一部として受け取る Web ページのための cookie、画像、JavaScript、そしてプラグインを、完全に無効にしてしまうことができます。

あなたが Web をブラウズする時には、こうした機能の一部、または全てを有効にしておくと便利であるかもしれませんが、メッセージの添付として送られてきた一枚の Web ページには必要ない機能かもしれません。

電子メールメッセージの中の cookie、画像、JavaScript、そしてプラグインを有効／無効にするための情報は、次のセクションをご覧ください：

私の個人情報を私の許可なく使われないようにするにはどうすればいいのでしょう？

あなたの情報を守る最も良い方法は、あなたが訪れるその Web サイトや、あなたが利用しているインターネットサービスのプライバシーポリシーを読むこと、そしてオンラインであなたの個人情報を提供することに注意することです。

インターネットは、公共のネットワークです。あなたが、名前、電話番号、住所、その他の個人情報をネットワークを経由して送った場合（Web ページや電子メール、あるいはその他の方法で）、誰かがその情報を途中で横取りすることも可能なのです。

以下に Web サイトのプライバシーポリシーに関して確認する項目を挙げました：

  • そのサイトは、どういう種類の個人情報を集めているのか？
  • そのサイトは、集めた情報をどのように利用するつもりなのか？
  • そのサイトは、他者と情報を共有するのか、また共有された情報に関して、私は何らかの選択肢があるのか？
  • 私は情報の管理や更新のために、そのサイトが関して集めた一部または全てのデータにアクセスすることができるのか？
  • そのサイトはどのようにして保存している情報を保護しているか？
  • 質問や問題が発生した場合、 Web サイトにどうコンタクトすればよいのか？

Web サイトのプライバシーポリシーを確認するための情報は、プライバシーポリシーを表示するをご覧ください。
