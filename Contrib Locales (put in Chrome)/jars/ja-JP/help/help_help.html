ヘルプウィドウを使う

このセクションでは、ヘルプウィンドウの使い方を説明します。− つまり、このテキストおよび左側に表示されているタブすべてについてです。

さらに詳しい情報は、Mozilla ヘルプへようこその下の部分に表示されているリンクをクリックしてください。

求めるトピックを探す

ヘルプウィンドウが表示されている場合、左側のフレームにあるタブをクリックして、様々な方法でヘルプの内容を見ることができます：

  • 目次は、主なトピックを一覧表示します。

    • トピックをクリックして、読んでください。ダブルクリックをすると、そのトピックの詳細が表示されます。

    探しているトピックが見つからない場合、［検索］または［索引］タブをクリックしてください。

  • 検索では、Mozilla ヘルプを検索できます。

    • 単語や熟語を入力し、［検索］を押して、関連するトピックを探してください。そして、読みたいトピックをクリックしてください。

    検索してもトピックが見つからない場合、入力する言葉を少なくしたり、異なる単語の組み合わせを試してください。

  • 索引アルファベット順にトピックが一覧表示されます。［訳注： 残念ながら、あいうえお順ではありませんし、日本語でもありません。］

    • 見出しをクリックして、そのトピックについての情報をご覧ください。

  • 用語集アルファベット順に定義が一覧表示されます。［訳注： 残念ながら、あいうえお順ではありません。］

    • 用語集にある言葉をクリックして、その定義を表示してください。

あなたが読んでいるヘルプトピックに関連した情報を表示するには、ちょうど Web ページを見ている時と同じように、ヘルプウィンドウ内のリンクをクリックしてください。

例えば、このリンクは、用語集にリンクされています： cookie　（このページに戻るには、［戻る］ボタンをクリックしてください。）

 

あなたの手順を繰り返す

ヘルプでのあなたの手順を繰り返すには、ヘルプウィンドウの右上角［訳注： 左上角？］にあるボタンをクリックしてください：

         ヘルプホーム
         戻る       進む

  • あなたが前に見たページを戻るには、［戻る］ボタンをクリックしてください。このボタンは、ちょうどブラウザの［戻る］ボタンと同じように働きます。
  • あなたが前に見たページに進むには、［進む］ボタンをクリックしてください。このボタンは、ちょうどブラウザの［進む］ボタンと同じように働きます。
  • ヘルプホームに戻るには、［ホーム］ボタンをクリックしてください。このボタンは、サポートオプションや Web ベースの情報源にアクセスしていても有効です。

指示に従ううちに、ヘルプウィドウが前面に表示できなくなってしまった場合、ウィンドウズツールバーの Mozilla Help をクリックすると、再び前面に表示できます。

または、あなたが行いたい操作を印刷することができます：

  • その時点でヘルプウィンドウに表示されているページ全体を印刷したい場合は、ヘルプウィンドウの右上角にある［印刷］ボタンをクリックしてください：


    •   印刷

  • ページの一部分を印刷する場合は、始めにクリックしてからドラッグして、印刷したい範囲を選択します。そして、印刷ボタンをクリックしてください。

 

ヘルプショートカットを使う

ヘルプウィンドウ内をナビゲートするのに、キーボードショートカットを利用することができます。詳細情報は、キーボードショートカットヘルプをご覧ください。

 

ヘルプボタンを使う

多くの Mozilla ウィンドウやダイアログボックスには、"ヘルプ" ボタンがついています。

［ヘルプ］ボタンをクリックして、表示されているウィンドウの詳細情報をご覧ください。
