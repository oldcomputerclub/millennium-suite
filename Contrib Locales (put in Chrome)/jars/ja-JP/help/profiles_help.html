プロファイルを管理する

インターネットを自宅と職場で使う場合、ブックマーク、設定、アドレス帳、電子メールアカウント、Sidebar の設定などを、別々に利用したいと思うでしょう。同じように、家族が同じブラウザソフトを共有したいけれど、インターネットの個人情報は分けておきたくなるかもしれません。

プロファイルマネージャは、異なったプロファイルを作成できるようにします。プロファイルには、ブックマーク、設定、電子メール設定などがそれぞれ入っています。ブラウザソフトを最初にインストールしたとき、標準のプロファイルが自動的に作成されます。追加のプロファイルを作成する場合、ブラウザの起動時にどのプロファイルを使うか選択することになります。

新しいプロファイルを作成する

プロファイルを作成するには：

  1. ブラウザを終了します。ブラウザソフトの Navigator、メール、その他のコンポーネントが実行中の間は、新しいプロファイルを作成することはできません。
  2. プロファイルマネージャを開きます：
    • Windows： ［スタート］メニューをクリックして、［プログラム］を選択します。そして［Mozilla］を開き、次に［Profile Manager］をクリックします。
      ［訳注： インストーラでなく、zip 版で導入した場合は、スタートメニューをクリックして「ファイル名を指定して実行」を選択し、「（格納フォルダ名） \mozilla.exe -ProfileManager」と指定します］
    • Mac OS： Mozilla Folder を開き、中の Mozilla Profile Manager をダブルクリックします。
    • Linux： コマンドラインで以下のように入力します： ./mozilla -profilemanager
  3. ［新しいプロファイルの作成］ をクリックします。プロファイルの説明を読んで ［次へ］ をクリックします。
  4. プロファイル名を入力します。あなたの本名や、"学校" のような、何のためにプロファイルに使うかに関連する名前といった、お好みの名前を入力してください。
  5. 新しいプロファイルを標準の場所に保存することを受け入れるか、［フォルダを選択］ ボタンをクリックし、保存したい場所まで進んでください。
  6. 言語パック（Language Pack）や地域ごとのコンテンツパック（Content Pack）がインストールされている場合、［地域の選択］ をクリックして、［言語／地域の選択］を設定します。
  7. ［完了］ をクリックします。

再度プロファイルマネージャが画面に現れます。一覧に新しいプロファイルが追加され、ハイライトされています。

 

プロファイルを削除する、名前を変える

通常使用しないプロファイルを削除したいときがあります。既存のプロファイルを削除したり、名前を変更するには：

  1. ブラウザを終了します。ブラウザソフトの Navigator、メール、その他のコンポーネントが実行中の間は、新しいプロファイルを作成したり、名前を変更することはできません。
  2. プロファイルマネージャを開きます：
    • Windows： ［スタート］メニューをクリックして、［プログラム］を選択します。そして［Mozilla］を開き、次に［Profile Manager］をクリックします。
      ［訳注： インストーラでなく、zip 版で導入した場合は、スタートメニューをクリックして「ファイル名を指定して実行」を選択し、「（格納フォルダ名） \mozilla.exe -ProfileManager」と指定します］
    • Mac OS： Mozilla Folder を開き、中の Mozilla Profile Manager をダブルクリックします。
    • Linux： コマンドラインで以下のように入力します： ./mozilla -profile
  3. プロファイルを削除するには、削除するプロファイル名を選択して、［プロファイルの削除］をクリックします。
  4. プロファイルの名前を変更するには、名前を変更するプロファイル名を選択して、［プロファイルの名前変更］をクリックします。後は指示に従います。
