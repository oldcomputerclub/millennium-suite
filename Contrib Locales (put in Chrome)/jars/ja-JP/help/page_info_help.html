ページの情報を表示する

ページの情報ダイアログボックスは、あなたが見ているページについて、様々な種類の情報を表示する、いくつかのタブで構成されています。このセクションでは、それぞれのタブで得られる情報についての概要を提供します。

その時点でブラウザで表示されているページのページの情報を表示するには、［表示］メニューを開き、［ページの情報］を選択します。あなたが表示したい情報に対応するタブをクリックしてください。

ページの情報内の［一般］、［フォーム］、［リンク］、および［メディア］タブによって表示される情報のほとんどは、そのページの HTML ソースから取得されます。この情報は、通常、Web 開発者やその他の専門家のみが関心を持つものです。

［ページの情報］によって表示されるタグを含めた、HTML に関する詳細情報は、 HTML 4.01 Specification （英文）をご覧ください。
［訳注： 日本語での情報は、こちらまたはこちらをご覧ください。］

［プライバシー］および［セキュリティ］タブには、Platform for Privacy Preferences（P3P） （英文）についての情報が含まれています。P3P とは、あなたが誰のページを見ているのか、およびそのページのセキュリティ状況を示すものです。

一般 （ページの情報タブ）

［表示］メニューから［ページの情報］を選択すると、［一般］タブが、あなたがブラウザで見ているページについての基本的な情報を表示します。

上の部分には、そのページの名前（もしあれば）および次の情報が表示されます：

  • URL：  そのページの Uniform Resource Locator、つまり、ブラウザウィンドウの上部付近のロケーションバーに表示される、標準化されたアドレスのことです。
  • レンダリング：  ブラウザが、そのページを表示するのに、Quirks（後方互換）モードまたは Standard（標準準拠）モードを使用しているのかを示します。Quirks モードは、完全に Web 標準には準拠していない、古いバージョンの Web ブラウザのためにデザインされた、古い Web ページによって使用される、非標準的な動作のことを示します。Standard モードは、仕様が定める標準に完全に準拠していることを示します。あなたのブラウザは、その Web ページに含まれる情報に従って、自動的に描画モードを選択します。
  • ソース：  そのページのソースコードがキャッシュされているかどうかを示します。
  • 文字コード：  その HTML ドキュメントのために使用されている文字コードセット。
  • サイズ：  そのファイルのサイズ（入手可能な場合）。
  • 更新日時：  そのページが最後に変更された日付（入手可能な場合）。
  • 有効日時：  そのページによって表示されている情報が無効になる日付。

ダイアログボックスの下の部分には、そのページによって指定されたメタタグを表示します。メタタグとは、そのページの一般的な説明、検索エンジンのためのキーワード、著作権情報などといった、ページによって表示される内容の種類についての情報のことです。

 

フォーム （ページの情報タブ）

［表示］メニューから［ページの情報］を選択して、［フォーム］タブを選択した場合、その時点でブラウザに表示されているページ内の全てのフォーム情報を見ることができます。

上の部分には、そのページ内のそれぞれのフォームが、HTML ソース内でどのように指定されているか、という基本的な情報を一覧表示します：

  • 名前：  フォームの名前（もしあれば）。
  • メソッド：  フォームによって取り込まれた情報を Web サーバーに送り返す際に使用される HTML メソッド。GET は、あなたが記入した値を、そのフォームを送信する Web サイトアドレスに添付します。POST は、Web サイトに値を、その Web サイト上にあるプログラムによって読み込めるようなパラメーターとして送信します。
  • フォームのアクション：  フォームが送信された時に呼び出されるプログラムの URL。

［フォーム］タブの上の部分に一覧表示されたフォームを選択した場合、下の部分に、フォームのそれぞれの要素が HTML ソース内で、どのように指定されているのかについての詳細情報を表示します：

  • ラベル：  その要素のラベル（HTML 内でそのようにタグがされていれば）。
  • フィールド名：  その要素の名前。
  • タイプ：  TEXT（テキストを送信する用）、RADIO（ラジオボタン用）、HIDDEN（送信されるが、スクリーン上には表示されない情報を保存する用）といった、その要素の入力タイプ。
  • 現在の値：  その要素の、その時点での値。例えば、テキスト要素のその時点での値は、その要素がその時点で含んでいるテキストのことです。

 

リンク （ページの情報タブ）

［表示］メニューから［ページの情報］を選択して、［リンク］タブを選択した場合、そのページで利用できる全てのリンクが一覧表示されます。それぞれのリンクについて、次の情報が表示されます：

  • 名前：  ブラウザ上にリンクとして表示されているテキスト。
  • アドレス：  そのリンクが指しているページの URL。
  • タイプ：  アンカー（HTML ドキュメント内の特定の場所へのリンク）またはフォーム送信といった、リンクのタイプ。

 

メディア （ページの情報タブ）

［表示］メニューから［ページの情報］を選択して、［メディア］タブを選択した場合、そのページに表示されている全ての画像やその他のメディア要素が一覧表示されます。

上の部分には、アドレス（URL）や種類といった、それぞれの要素の基本的な情報が一覧表示されます。

［メディア］タブの上部に一覧表示されたメディア要素を選択すると、下の部分に、その要素の名前、サイズ、または寸法といった、入手可能な情報が表示されます。

さらに、ダイアログボックスの下の部分で、選択された要素を見ることができます。より大きな画像を見るためには、［ページの情報］ダイアログボックスの右下隅をクリックして、そのままドラッグしてください。

メディア要素を個別のファイルとして保存するには：

  • 名前を付けて保存： 保存したい要素を選択して、［名前を付けて保存］をクリックし、そして保存したい場所を指定します。

 

セキュリティ（ページの情報タブ）

［表示］メニューから［ページの情報］を選択して、［セキュリティ］タブを選択した場合、あなたが見ている Web ページの検証および暗号についての情報が表示されます。上の部分ではその Web サイトの出所が検証されているかを、そして、下の部分では、そのページがブラウザによって受け取られた際に、そのページが暗号化されているかどうかを示します。

この［セキュリティ］タブは、ブラウザウィンドウの右下隅にある鍵アイコンをクリックすることで、直接開くことができます。

［セキュリティ］タブの上の部分には、次のメッセージのどれかが表示されます：

  • Web サイトは特定できません。検証されていません。： あなたが見ている Web サイトは、サイトを証明する証明書を提示しませんでした。そのため、証明書マネージャは、そのページを特定できませんでした。可能性は低いものの、その Web サイトは偽装されている可能性があります。
  • Web サイトを特定しました。検証されています。： 証明書マネージャがこの Web サイトを特定するために使用する証明書が、認証局（CA）から発行されており、信頼できるサイトとされています。その Web サイトは信頼できると考えて良いでしょう。

［セキュリティ］タブの下の部分には、以下のメッセージが表示されます：

  • 接続は暗号化されていません： あなたのコンピュータからその Web サイトへ向けて送られた情報や、その Web サイトからあなたのコンピュータに向けて送られた情報を、他者が見ることができる可能性があります。しかし、実際に誰かが盗み見ている可能性は、高くありません。
  • 接続は暗号化されています： 一般的に、暗号化された接続の強度は、ビット数で測られる、暗号に使用されるキーの長さに依存します。キーの長さが長い程、暗号化は強くなります。つまり、許可のない者がその暗号化された情報を解読するのは、より困難になります。

    • ［ページの情報］ウィンドウでは、次の三つのうちの一つで、暗号の強度を示します：

    • 高程度な品質の暗号： 最低でも 128 ビットキーを使用した、利用できるもっとも強固な暗号。
    • 中程度な品質の暗号： 56 または 64 ビットキーを使用した、低品質な暗号よりは多少強固な暗号。
    • 低程度な品質の暗号： 40 ビットキーを使用した、利用できるもっとも弱い暗号。

    ほとんどの Web サイトは、高程度な品質の暗号をサポートしています。弱いタイプの暗号をサポートする古い Web サイトを見ている場合、あなたのコンピュータからその Web サイトへ向けて送られた情報や、その Web サイトからあなたのコンピュータに向けて送られた情報を、他者が見ることができる可能性があります。しかし、実際に誰かが盗み見ている可能性は、高くありません。

暗号化がされていない、または強固な暗号化がされていないというのは、あなたが、クレジットカード番号といった、機密情報を送信している時や表示している時にのみ、問題になります。

ブラウザウィンドウの右下隅の鍵アイコンの状況を見ることで、Web ページの暗号化状況を簡単に確認することができます。詳細情報は、Web ページのセキュリティをチェックするをご覧ください。
