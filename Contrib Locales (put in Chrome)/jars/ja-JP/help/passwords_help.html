パスワードマネージャ

パスワードマネージャで次のことができます：

  • ユーザー名やパスワードを保存する： このチェックボックスを選択して、パスワードマネージャを有効にしてください。パスワードマネージャは、あなたのユーザー名とパスワードを適時尋ね、そのような情報が要求された時に、あなたのために自動的に入力します。パスワードマネージャの機能を停止するためには、同じチェックボックスのチェックをはずします。
  • 保存されたパスワードを管理する： このボタンをクリックすると、ユーザー名やパスワードを保存したくないサイトで、既に保存されたパスワードやサイトの情報を表示します。

個々のサイトを除いたり、保存されたパスワードを表示したり管理したりするなど、パスワードマネージャを利用する際の詳細情報はパスワードマネージャを利用するをご覧ください。

 

暗号化するか、隠蔽化するか

パスワードマネージャやフォームマネージャを利用してパスワードや個人情報を保存している場合は、これらの機密情報があなたのコンピュータ上にファイルとして保存されます。このファイルは、侵入者が読み取ることは困難ですが、不可能ではありません。このような方法での情報の保存を、"隠蔽化" と言います。この方法は標準の設定で、パスワードマネージャおよびフォームマネージャによって保存された情報について当てはまります。

より安全な保存方法のためには、ファイルを暗号化して保護する方法があります。暗号化をすることで、権限のない人が、あなたの保存された機密情報を見ることがより困難になります。

  • 機密情報を保存する際に暗号化機能を利用する： このチェックボックスをチェックして暗号化機能を利用可能にします。チェックをはずすとこの機能が使えなくなります。

前もってマスターパスワードが設定されていない場合、新に設定するかどうか尋ねられます。設定する場合は、画面に表示される指示に従ってください。

保存された機密データを暗号化するか隠蔽するかは、より良いセキュリティと利便性とのトレードオフとなります：

  • 暗号化を利用する場合、定期的にマスターパスワードを入力する必要があります。これは不便なことになる可能性があります。（マスターパスワードがどれだけ頻繁に要求されるかどうかを管理するための情報は、プライバシーとセキュリティ設定 - マスターパスワードにある、マスターパスワードの有効期限についての解説をご覧ください。）
  • 隠蔽化を利用する場合、マスターパスワードを設定する必要は全くありません（身分証明のために証明書を使用していない限り）。ただし、他の人があなたのコンピュータにアクセスして、あなたのパスワードを盗むのは、暗号化した場合に比べて簡単になります。

より詳しい情報は、保存された機密情報を暗号化するをご覧ください。

 

パスワードマネージャ

このセクションでは、保存されたあなたのパスワードを管理するためのパスワードマネージャダイアログボックスの使い方を解説します。ダイアログボックスが表示されていない場合は、次の手順に従ってください：

  1. ［表示］ メニューから ［設定］ を選択します。
  2. ［プライバシーとセキュリティ］ カテゴリの中から、［パスワード］ を選択します。 （サブカテゴリが開いていない場合は、［プライバシーとセキュリティ］をダブルクリックしてリストを開いてください。）
  3. ［保存されたパスワードを管理する］をクリックします。

パスワードマネージャには二つのタブがあります：

  • 保存されたパスワード： このタブをクリックすると、パスワードマネージャが保存したあなたのユーザー名やパスワードの一覧が表示されます。つまり、パスワードマネージャが、ログイン情報を保存するかどうかを尋ねた問いに "はい" を選択したサイトが表示されます。

    右側の欄にはそれぞれのサイトで利用するユーザー名が表示されます。パスワードが暗号化された形で保存されている場合、ユーザー名の後に "（暗号化）" と表示されます。（保存されたパスワード自体は表示されません。）

    一覧から見出しを削除した場合、保存されたユーザー名とパスワードは破棄され、次にそのサイトにログインするには手動で情報を入力する必要があります。

  • 保存されないパスワード： このタブをクリックして、パスワードマネージャの問いに "決してしない" と選択したサイトの一覧が表示されます。

    サイトがこのリストに含まれている場合、このサイトをログオンする際に、手動でユーザー名とパスワードを手動で入力しなければなりません。

    この一覧から見出しを削除した場合、次にそのサイトをログオンする際に、パスワードマネージャがユーザー名とパスワードを保存するかどうか、再度問いを表示します。

表示するタブに関わらず、次の手順に従って一覧から見出しを削除することができます：

  • 削除： 削除したい見出しを選択して、［削除］をクリックします。
  • 全て削除： このボタンをクリックすると、表示しているタブにある全ての見出しを削除します。

パスワードマネージャについての詳細情報は、パスワードマネージャを利用するをご覧ください。

マスターパスワードを変更する

マスターパスワードは、あなたのセキュリティデバイスを保護します。セキュリティデバイスとは、 暗号鍵や証明書のような、あなたの身元証明に関連した機密情報を保存するソフトウェア、 またはハードウェアのことです。

例えば、ブラウザはビルトインのソフトウェアセキュリティデバイスを持っており、 コンピュータの設定がなされていれば、 スマートカードのような外部のセキュリティデバイスを利用することもできます。

ブラウザにビルトインされているソフトウェアセキュリティデバイス用のマスターパスワードはまた、パスワードマネージャやフォームマネージャによって保存された、電子メールパスワード、 Web サイトパスワードやその他のデータといった、保存された機密情報をも保護します。

  • パスワードを変更する： マスターパスワードを設定または変更するには、このボタンをクリックします。マスターパスワード変更ダイアログボックスの使い方についての情報は、マスターパスワードを変更するをご覧ください。

 

マスターパスワードの有効期限

新しいマスターパスワードをはじめて設定した後に、新しくブラウザを立ち上げた時にのみ、ユーザー名やパスワード、保存されたフォームデータ、または個人証明書といった個人情報にアクセスするために、マスターパスワードを、ブラウザが必要とするかどうかを尋ねられます。

ブラウザがどの程度の頻度でマスターパスワードを要求するか管理することができます：

  • 初めて必要になった時に： この設定は標準で選択されていますが、ブラウザを立ち上げた後、初めてプライベートキーデータベースにアクセスする必要ができた時にのみ、マスターパスワードを要求します。
  • 要求される度に： この設定では、保存された個人情報にアクセスするたびに必ず、ブラウザはマスターパスワードを要求するように設定できます。この設定は、最も高いレベルの保護です。
  • __ 分間以上利用されなかった場合： この設定では、あなたの個人情報にアクセスする必要がある場合で、最後に利用されてから指定された時間以上経っている場合に、マスターパスワードを要求します。

 

マスターパスワードをリセットする

警告： マスターパスワードをリセットした場合、パスワードマネージャやフォームマネージャによって保存された全ての暗号化された Web パスワード、電子メールパスワード、およびフォームデータが、完全に消去されます。またソフトウェアセキュリティデバイスに関連した個人証明書も消去されます。

保存された情報の暗号化を有効にした場合にのみ、すでに保存されたデータが失われてしまうことがあります。暗号化を有効／無効にすることについての情報は、暗号化を有効／無効にするをご覧ください。

マスターパスワードをリセットではなく変更するには、［マスターパスワード］設定パネルの［パスワードの変更］ボタンをクリックしてください。

［パスワードの変更］ボタンでマスターパスワードを変更する場合には、以前のパスワードが必要となります。

あなたが本当にマスターパスワードをリセットして、暗号化されたあなたの個人情報や証明書を完全に消去したい場合は：

  • マスターパスワードをリセットする： ソフトウェアセキュリティデバイス用のマスターパスワードをリセットするにはこのボタンをクリックします。詳細情報は、マスターパスワードをリセットするをご覧ください。

 

マスターパスワードを変更する

このセクションでは、［マスターパスワードの変更］ダイアログボックスについて解説します。ダイアログがまだ表示されていない場合は、以下の手順を実行してください：

  1. ［表示］ メニューから ［設定］ を選択します。
  2. ［プライバシーとセキュリティ］ カテゴリの中から、［マスターパスワード］ をクリックします。 （サブカテゴリが開いていない場合は、［プライバシーとセキュリティ］をダブルクリックしてリストを開いてください。）
  3. ［パスワードの変更］をクリックします。

マスターパスワードは、暗号鍵や証明書といったあなたの個人情報に関連した機密情報を保存するソフトウェア、またはハードウェアデバイスである、セキュリティデバイスを保護します。

例えば、ブラウザにはビルトインのソフトウェアセキュリティデバイスがありますし、コンピュータの設定によってはスマートカードのような外部のセキュリティデバイスを利用することもできます。

ブラウザのビルトインソフトウェアセキュリティデバイス用のマスターパスワードはまた、あなたのマスターキーを保護します。あなたのマスターキーは、電子メールパスワード、Web サイトパスワードや、パスワードマネージャやフォームマネージャによって保存されたデータのような機密情報を暗号化するために利用されます。

［マスターパスワードの変更］ダイアログボックスで以下の情報を入力します：

  • セキュリティデバイス： それぞれのセキュリティデバイスに個別のパスワードが必要です。例えば、スマートカードを利用して個人証明書を保存している場合、それぞれに対して別のマスターパスワードを設定しなければなりません。複数のセキュリティデバイスが利用できる場合、［マスターパスワードの設定］ダイアログボックスの上部にあるドロップダウンリストで、どのデバイスのパスワードを変更するか選択できます。

  • 現在のパスワード： 既に利用しているマスターパスワードを変更するには、始めに現在使用しているパスワードを入力しなければなりません。このパスワードが正しく入力されないと、［OK］をクリックしたときに "現在の正しいマスターパスワードが入力されませんでした" というメッセージが表示されます。このメッセージが表示されたら、再度あなたが現在使用しているパスワードを入力してください。

  • 新しいパスワード： このフィールドに新しいパスワードを入力してください。
  • 新しいパスワード（再度）： もう一度新しいパスワードを入力します。ここで始めに入力したパスワードと完全に同じパスワードを入力しないと、［OK］ボタンは選択できないままです。この場合は、新しいパスワードを入力し直してください。

あなたのマスターパスワードを利用できたり、予測できるような人があなたのコンピュータを利用した場合、その人はあなたになりすまして Web サイトにアクセスできる可能性があります。これは非常に危険な場合があります。− 例えば、インターネット上で投資情報を管理している場合です。

ですから、想像することが難しいマスターパスワードを設定することが大切になります。パスワード品質メーターを利用して、あなたが入力したパスワードが、長さや大文字／小文字／数字、そしてシンボルの使い方の観点からあなたのパスワードの質を大まかに確かめることができます。ただし、誰にもあなたのパスワードが分からないということを保証するものではありません。

さらに詳しいガイドラインとして、オンラインドキュメントの 良いパスワードを設定するには をご覧ください。

あなたのマスターパスワードを安全な場所に保存しておくことも大切です。− ただし、簡単に他の人がアクセス できない ような場所に、です。もしこのパスワードを忘れると、あなたのコンピュータに保存されているパスワードや証明書を必要とする Web サイトといった、大切な情報にアクセスできなくなる可能性があります。

 

マスターパスワードをリセットする

このセクションでは［マスターパスワードをリセット］ダイアログボックスについて解説します。このダイアログボックスが表示されていない場合は、次の手順に従ってください：

  1. ［編集］メニューを開いて、［設定］を選択します。
  2. ［プライバシーとセキュリティ］カテゴリの［マスターパスワード］をクリックします。 （サブカテゴリが開いていない場合は、［プライバシーとセキュリティ］をクリックしてリストを開いてください。）
  3. ［パスワードをリセット］をクリックします。

警告： マスターパスワードをリセットした場合、パスワードマネージャやフォームマネージャによって保存された暗号化された Web パスワード、電子メールパスワード、そしてフォームデータは全て完全に消去されてしまいます。また ソフトウェアセキュリティデバイスに関連した個人証明書もまた消去されてしまいます。

保存された情報の暗号化を有効にした場合にのみ、すでに暗号化されたパスワードやフォームデータが失われてしまうことがあります。暗号化を有効／無効にすることについての情報は、暗号化を有効／無効にするをご覧ください。

マスターパスワードを覚えていて、それを変更したい場合は、個人情報を失う危険を冒すことなく変更することができます。マスターパスワードリセットの警告表示がされていて、マスターパスワードをリセットするのではなく、変更したい場合は、［キャンセル］ をクリックしてマスターパスワード設定パネルに戻ってください。そして ［パスワードを変更］をクリックしてください。詳細については、マスターパスワードを変更する をご覧ください。

マスターパスワードをリセットするというのは、どうしてもマスターパスワードを思い出せなくなってしまった場合の、最後の手段です。リセットによって、どの程度深刻な影響があるかは、あなたのマスターパスワードがどの程度個人情報を保存していたかによります。

マスターパスワードをリセットしても、新しいパスワードは作成されません。その替わり、以前のマスターパスワードが保護していた全てのデータを削除します。次回、ブラウザが個人情報を保存する必要があるときに、新しいマスターパスワードを指定するように尋ねられます。

マスターパスワードをリセットした後、将来前もってデータを入力できるように、個人情報を再保存したいかもしれません。例えば、ブラウズしている時に、パスワードマネージャが Web サイトおよび電子メールパスワードを保存したい場合です。

フォームマネージャが自動的に全てのフォームを入力できるようになるまで、自分でデータを入力する必要もあります。さらにソフトウェアセキュリティデバイスに関連する全ての個人証明書は完全に消去されます。 そのため新しい証明書を取得しなければなりません。

スマートカード利用者への注意： スマートカードは独自のマスターパスワードを使用しています。 スマートカードのマスターパスワードは （個人証明書などの） スマートカード上のデータしか保護していません。 多くの場合、スマートカードのマスターパスワードを変更することができます （あなたがマスターパスワードを覚えていることが前提です）。しかしリセットはできません。
